# 💳 CreditWise – AI-Powered Loan Approval & Credit Health Prediction

CreditWise is a Machine Learning-based web application that predicts loan approval while also analyzing a user's credit behavior and providing personalized suggestions to improve their credit health.

This project was developed as my End Semester Machine Learning Project using Python, Flask, and Scikit-learn.

---

## 🚀 Features

- 🤖 Loan Approval Prediction using Machine Learning
- 📊 Credit Health Analysis
- 💡 Personalized Credit Improvement Suggestions
- 📈 Rule-Based Validation for unrealistic loan requests
- 🎨 Modern Responsive User Interface
- ⚡ Fast predictions using a trained ML model

## ⚙️ Workflow

1. User enters financial and credit information.
2. Flask backend processes the input.
3. Trained Machine Learning model predicts loan approval.
4. Credit behavior is analyzed.
5. Personalized suggestions are generated.
6. Final prediction and Credit Health Score are displayed.

## ▶️ Installation

Clone the repository

```bash
git clone https://github.com/your-username/CreditWise.git
```

Move into the project folder

```bash
cd CreditWise
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
python app.py
```

Open your browser and visit

```
http://127.0.0.1:5000
```
